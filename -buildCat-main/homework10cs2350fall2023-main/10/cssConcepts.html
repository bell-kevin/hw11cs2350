The CSS Box Model

The CSS box model consists of four parts: content, padding, border, and margin.

Content: This is the innermost part of the box and holds the actual content, such as text, images, or other media. The size of the content area is determined by the width and height properties you set in your CSS.

Padding: Padding is the space between the content and the element's border. You can set padding using the padding property. Padding adds space inside the element, increasing its overall size. It can be uniform on all sides or specific to each side (e.g., padding-top, padding-right, padding-bottom, and padding-left).

Border: The border is the line or boundary that surrounds the padding and content of an element. You can define the border's width, style, and color using properties like border-width, border-style, and border-color. The border adds to the overall size of the element.

Margin: Margins are the spaces outside of the border that separate an element from other elements on the web page. You can set margins using the margin property. Margins do not have a background color or border and are used to create space between elements. Like padding, margins can be uniform or specific to each side (e.g., margin-top, margin-right, margin-bottom, and margin-left).

Position

The CSS position property allows you to control the placement of elements on a web page.

Relative Position

When the position property is set to relative, an element's position is adjusted relative to its normal position in the document flow.

Absolute Position

When the position property is set to absolute, an element is positioned relative to its nearest positioned ancestor. If no ancestor has a position other than the default, it will be positioned relative to the initial containing block (usually the viewport).
